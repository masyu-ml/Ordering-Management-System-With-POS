# Ordering Management System with POS

A simple **Point-of-Sale (POS)** and ordering management system built with Python.  
This project allows managing products, orders, and customers efficiently.

## Features

- Add, update, and delete products
- Manage customer orders
- Track inventory
- Generate sales reports
- Simple user interface for POS operations
- Analytics and Dashboard

## Tech Stack

- **Language:** Python  
- **Framework:** FastAPI 
- **Database:** MySQL 
- **Frontend:** PYQT5
